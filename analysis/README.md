# Dossier analysis

Ce dossier accueille les scripts et notebooks d'analyse des réponses collectées.

## Fichiers prévus

- `explore.py` — chargement et exploration de `data/responses.csv`
- `profiles.py` — statistiques sur la distribution des profils
- `correlations.py` — corrélations entre scores et catalogue de jeux

## Lancer un script

```bash
uv run python analysis/explore.py
```
